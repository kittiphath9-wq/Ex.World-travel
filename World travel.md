# World Travel

## 1. ความเป็นมา
World Travel เป็นแพลตฟอร์มท่องเที่ยวที่ออกแบบมาเพื่อให้นักเดินทางสามารถค้นหาสถานที่ท่องเที่ยวที่น่าสนใจทั่วโลก พร้อมทั้งดูข้อมูล คำแนะนำ และรีวิวจากผู้ใช้งานคนอื่นๆ

## 2. วัตถุประสงค์
- เพื่อให้นักเดินทางสามารถค้นหาสถานที่ท่องเที่ยวที่น่าสนใจทั่วโลก
- เพื่อให้นักเดินทางสามารถดูข้อมูล คำแนะนำ และรีวิวจากผู้ใช้งานคนอื่นๆ
- เพื่อให้นักเดินทางสามารถวางแผนการเดินทางของตนเองได้อย่างสะดวกสบาย
- เพื่อให้นักเดินทางสามารถแบ่งปันประสบการณ์การเดินทางของตนเองกับผู้อื่น

## 3. คุณสมบัติของระบบ
### 3.1 การค้นหาสถานที่ท่องเที่ยว
- ค้นหาสถานที่ท่องเที่ยวตามชื่อ
- ค้นหาสถานที่ท่องเที่ยวตามหมวดหมู่
- ค้นหาสถานที่ท่องเที่ยวตามภูมิภาค
- ค้นหาสถานที่ท่องเที่ยวตามงบประมาณ
- ค้นหาสถานที่ท่องเที่ยวตามช่วงเวลา

### 3.2 ข้อมูลสถานที่ท่องเที่ยว
- ชื่อสถานที่ท่องเที่ยว
- คำอธิบายสถานที่ท่องเที่ยว
- รูปภาพสถานที่ท่องเที่ยว
- วิดีโอสถานที่ท่องเที่ยว
- ที่อยู่สถานที่ท่องเที่ยว
- ข้อมูลการเดินทางไปยังสถานที่ท่องเที่ยว
- ข้อมูลที่พักในบริเวณสถานที่ท่องเที่ยว
- ข้อมูลร้านอาหารในบริเวณสถานที่ท่องเที่ยว
- ข้อมูลกิจกรรมในบริเวณสถานที่ท่องเที่ยว

### 3.3 รีวิวสถานที่ท่องเที่ยว
- คะแนนสถานที่ท่องเที่ยว
- ความคิดเห็นเกี่ยวกับสถานที่ท่องเที่ยว
- รูปภาพสถานที่ท่องเที่ยว
- วิดีโอสถานที่ท่องเที่ยว
- วันที่รีวิว
- ชื่อผู้รีวิว

### 3.4 วางแผนการเดินทาง
- เพิ่มสถานที่ท่องเที่ยวในแผนการเดินทาง
- กำหนดวันเดินทาง
- กำหนดเวลาเดินทาง
- เพิ่มคำแนะนำส่วนตัว
- แบ่งปันแผนการเดินทางกับผู้อื่น

### 3.5 โปรไฟล์ผู้ใช้
- ชื่อผู้ใช้
- รูปโปรไฟล์
- อีเมล
- รหัสผ่าน
- ประวัติการเดินทาง
- รายชื่อสถานที่ท่องเที่ยวที่ชื่นชอบ

## 4. เทคโนโลยีที่จะใช้
- **Frontend**: React
- **Backend**: Node.js
- **Database**: MongoDB
- **Authentication**: JWT
- **Deployment**: Vercel
